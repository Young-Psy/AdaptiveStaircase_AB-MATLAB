# AdaptiveStaircase_AB-MATLAB
本项目将自适应阶梯程序（Adaptive Staircase Procedure）与注意瞬脱范式（Attentional Blink Paradigm）相结合，旨在精确测量注意滞留时长（Attentional Dwell Time）。 实验同时支持有节律（Rhythmic）与无节律（Arrhythmic）两种条件，完整包含以下模块：  被试信息采集（编号、年龄、性别等基本信息） 高精度视觉刺激呈现（基于 PTB / Psychtoolbox 后端实现毫秒级时序控制） 实时按键反应收集与数据记录 实验数据以 .mat 格式（MATLAB / Octave 兼容）自动保存，便于后续统计分析
